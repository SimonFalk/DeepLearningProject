Here are the datasets and python code for the submitted paper: 
MindID: Person Identification from Brain Waves through Attention-based Recurrent Neural Network

More comments and details will be added later.

MindID.py is the Tensorflow based python code.

EID-M.mat, EID-S.mat are two local datasets, please find more details in the paper.

The public dataset EEG-S mentioned in paper is a subset of EEG_ID_label6.mat. The generation of EEG-S is provided in the python code.
